Создать по пять различных уровней для каждого заднего фона.
Все фоны находяться в папке World - Tiles - Backgrounds. Каждый background имеет
свое название. На каждом уровне нужно создать слой Background layer
и настроить его должным образом. Поставить Speed y на 1 и включить галочку на
Horizontal Tile И Vertical Tile (Можно просто копировать уровни, а потом просто
заполнять их. Например Room1 имеет все необходимые слои). Каждые объекты и тайлы
должны бать на своем слое.

Слои:
Все необходимые слои с нужной иерархией уже созданы в комнате Room1.

Структура уровня:
Все уровни должны иметь одлин стиль, но можно немнго их смешивать
(на слое CollisionTiles должны находится все тайлы, из которых состоят уровни).
И на одном уровне может быть только либо зеленые тона, либо розовые, либо оранжевые.
Но они должны чередоваться каждые пять уровней.

В начале уровня всегда должен быть obj_player и obj_beginning, а в конце,
обязательно должен стоять obj_end. Они должны находиться на слое Environment.
Также на уровне (не важно где, но на слое Menegers должен распологаться объект
obj_game_manager). Также только в одном самом первом уровне должен находиться
obj_main_manager. На уровне также должны быть враги из папки Enemies и ловушки
из папки Traps.

Каждый последующий уровень должен быть дольше и сложнее предыдущего. Более
сложные уровни должны имет больше более сильных врагов и ловушек. Каждый пятый
уровень должен быть самым сложным из предыдущих четырех. И должен имет в
себе всех врагов и все ловушки, которые были в предыдущих четырех.

Также, время от времени, на уровнях должны распологаться obj_checkpoint.
Не слишком часто и только один за раз. В основном, они должны распологаться только
после каких-то сложных мест. Находится в папке Characters - Player - Checkpoints.

На каждом уровне должный распологаться также фрукты из папки Fruits. на одном
может быть любое количество лобого типа фрукта, но до 50 штук всех фруктов.




Враги:
Каждый враг должен распологаться на слое Enemys.

Одинаковые враги одного типа могут находиться лишь в двух уровнях подряд(не
считая каждый пятый уровень). На одном уровне могут присутствовать несколько
врагов разных видом. Всего врагов 20, так что нужно их всех
распределить по уровням. Каждый враг должен появлятся на уровне только своей
сложности. Далее про сложности врагов:

Сложности врагов:

Легкие (Должны присутствовать только в самых первых и легких уровнях):

Ckicken
Blue Bird (может появляться для того, чтобы игрок смог пробраться через
пропасть к секретке)

Нормальные (Должны присутствовать в более сложных и поздних уровнях):

Bat

Rino

Сложные (Должны присутствовать в самых поздних уровнях):

Все враги здесь распределены в порядке своей силы и мощи.

Особенности врагов:
Bee - при ее создании нужно в Creation Code указывать:
xx = 864 - позиция x точки, вокруг котороый она летает;
yy = 288 - позиция y точки, вокруг котороый она летает;
r = point_distance(x, y, xx, yy) - радиус (не изменять, так и написать);




Ловушки:

Каждая ловушка должна распологаться на слое Traps.

obj_falling_platform должен помогать игроку преодолеть расстояние, которое он
преодолеть не может (пропасть или забраться на верх) преодолеть не может  (пропасть или забраться на верх).

В объектах obj_platform_brown и obj_platform_grey в Creation Code должно быть
прописано:

left_path_x = 0; - крайняя левая x точка его пути
right_path_x = room_width; - крайняя правая x точка его пути

или

top_path_y = 0; - крайняя верхняя y точка его пути
bottom_path_y = room_height; - крайняя нижняя y точка его пути

и

dir_y = 0; - направление куда платформа сама по себе поедит по y
координате (1 - вниз, -1 - вверх)

или

dir_x = 0; - направление куда платформа сама по себе поедит по x
координате (1 - вправо, -1 - влево)

На их пути (куда они едут должны распологаться obj_platform_chain)

obj_platform_brown - едит, когда игрок на ней
obj_platform_grey - едит сама по себе


obj_arrow - подбрасывает игрока в воздух, весит в воздухе
obj_block - простол разрушаемый блок (если ударить по нему головой)
obj_fire - наносит игроку урон огнем, стоит на земле
obj_rock_head - ездит по квадрату


obj_saw - ездит по намеченному пути, так же как и платформы:
left_path_x = 0;
right_path_x = room_width;
top_path_y = 0;
bottom_path_y = room_height;

На ее пути также должны быть obj_saw_chain.


obj_spike_head - ездит по квадрату


obj_spiked_ball - иммет два режима (mode = false - вращается по кругу, mode = true - вращается полукругом)


obj_spikes - самая простая ловушка, наносит урон игроку при касании


obj_trampoline - подбрасывает игрока в воздух, помогая взобраться на
высокие уступы

(Позволяет игроку скользить по нему).
Не должны распологаться впритык к другой стене!!!!
На слое TrapsLayerSand должен распологаться песок (Оранжевый)
Они должны находиться на уровне вместо тайлов со слоя CollisionTiles,
а не поверх других тайлов!!!
